L'importance d'une variable est calculée par Permutation: on mélange aléatoirement les valeurs de la variable, on refait les prédictions, et on mesure l'augmentation d'erreur. Une erreur plus grande = variable plus importante.

  • Variables de Haut Rang: Impactent beaucoup les prédictions du modèle
  • Variables de Bas Rang: Peu d'impact sur les prédictions
  • Variables Catégorisées 📁: Caractéristiques distinctives (brand, carburant, etc.)
  • Variables Numériques 📊: Mesures continues (âge, kilométrage, prix, etc.)

Rapport généré automatiquement par Entity Embeddings Pipeline v3

Clustering: K-Means (K optimal trouvé par Silhouette Score) sur 30 dimensions (embeddings structurels)

Visualisation: t-SNE 2D
